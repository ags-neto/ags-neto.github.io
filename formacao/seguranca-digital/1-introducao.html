Sessão 1: Introdução à Segurança Digital

Apresentação e Objetivos

Segurança digital é o conjunto de práticas para proteger a nossa identidade, dados e dispositivos quando usamos a internet. Nesta primeira sessão vamos perceber os conceitos básicos e começar a criar hábitos seguros.

Endereço IP

O IP (Protocolo de Internet) é um número que identifica um dispositivo numa rede. É essencial para que a informação chegue ao destino correto.

1. Tipos de IP

  • IP Público: Identifica a tua ligação na Internet. É atribuído pelo fornecedor de Internet (ISP) e é visível para sites e serviços.
  • IP Privado: Usado dentro da rede doméstica ou empresarial.
  • IPv4: Formato mais comum, 4 grupos de números (ex.: 192.168.0.1).
  • IPv6: Formato mais recente, suporta muito mais endereços (ex.: 2001:0db8:85a3:fe01:0000:0000:0000:0000)

2. IP de Rede vs IP de Dispositivo

  • O IP da rede (público) é partilhado por todos os dispositivos ligados ao mesmo router.
  • Cada dispositivo recebe um IP privado único dentro da rede.

3. Que dispositivos têm IP?

  • Computadores, portáteis e telemóveis.
  • Tablets e consolas de jogos.
  • Smart TVs (uma TV normal não tem IP).
  • Equipamentos IoT (câmaras, assistentes virtuais, impressoras, etc.).

4. Como alguém pode obter o teu IP?

  • Quando visitas um site (o servidor regista o IP público).
  • Em chamadas P2P (ex.: jogos online, torrents).
  • Ao clicar num link ou imagem que regista pedidos no servidor.

5. O que podem fazer com o teu IP?

  • Determinar a localização aproximada (cidade/região).
  • Bloquear-te o acesso a um serviço (banimento).
  • Lançar ataques DDoS para sobrecarregar a tua ligação.
  • Tentar explorar vulnerabilidades no teu router (se não estiver protegido).

6. Como proteger e esconder o IP?

  • Usar uma VPN para mascarar o IP público.
  • Evitar Wi-Fi público sem proteção (usar VPN ou rede móvel).
  • Manter router e firewall atualizados.

7. Exercícios práticos

  1. Ver o teu IP e localização em What Is My IP Address.
  2. Ligar uma VPN e verificar como o IP muda.
  3. Listar todos os dispositivos ligados à tua rede usando Nmap Online ou app de gestão do router.

[15:00 - 25:00] Identidade Digital

Tudo o que publicamos, pesquisamos ou partilhamos constrói a nossa identidade digital. Uma vez online, é muito difícil apagar algo completamente.

  • Evitar publicar dados pessoais (morada, escola, telefone).
  • Pensar antes de partilhar fotos ou vídeos.
  • Desativar geolocalização em fotos.

Exercício: pesquisar o próprio nome no Google e analisar o que aparece.

[25:00 - 40:00] Segurança em Redes Sociais

As redes sociais recolhem dados para mostrar conteúdo e anúncios. É importante configurar a privacidade.

  • Revisão de permissões e amigos/contactos.
  • Evitar aceitar pedidos de estranhos.
  • Não clicar em links suspeitos no chat.

Links úteis para testar:

  • Quiz de links suspeitos
  • Ver se o meu email foi comprometido

[40:00 - 55:00] Noções de Segurança Técnica

  • HTTPS vs HTTP — sempre preferir sites com HTTPS.
  • Wi-Fi público — usar apenas com VPN e evitar transações sensíveis.
  • Atualizações — manter sistemas e apps atualizados.

Exercício: abrir dois sites, um com HTTP e outro com HTTPS, e ver o ícone do cadeado no browser.

[55:00 - 75:00] Passwords Seguras

Uma boa password é longa, única e difícil de adivinhar.

Fórmula exemplo: [PalavraBase]+[Ano]+[2 letras do site]

Exemplo: SolAzul2024FB (Facebook) e SolAzul2024GM (Gmail).

Ferramenta para testar força da password: Password Strength Test

[75:00 - 90:00] Desafio da Sessão

  1. Descobrir o teu IP público.
  2. Pesquisar o teu nome no Google e listar 3 resultados.
  3. Fazer o quiz de phishing e anotar pontuação.
  4. Alterar uma password fraca para uma fórmula segura.
