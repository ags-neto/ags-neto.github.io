Sessão 1: Como Funciona a Internet

1. Introdução

A Internet é uma rede mundial que liga milhões de dispositivos, computadores, smartphones, servidores, permitindo a troca de informação em tempo real. Embora pareça algo "no ar", a Internet depende de infraestruturas físicas e de regras invisíveis que organizam essa comunicação.

2. A Estrutura Física da Internet

2.1 Cabos Submarinos

A Internet atravessa oceanos através de cabos submarinos que ligam continentes. Estes cabos, que podem ter mais de 30 mil quilómetros, transportam quase todo o tráfego mundial de dados. Sem eles, não haveria ligação global.

Podes explorar o mapa dos cabos em Submarine Cable Map.

2.2 Servidores e Data Centers

Os websites, vídeos e serviços online vivem em computadores chamados servidores. Estes servidores estão alojados em data centers, edifícios preparados para manter milhares de máquinas sempre ligadas, com segurança e arrefecimento.

2.3 Como a Internet Chega a Tua Casa

Os fornecedores de Internet (ISP) ligam-te a esta rede global através de cabos de fibra ótica, ADSL ou cabo coaxial. O router distribui essa ligação dentro da tua casa, seja por Wi-Fi ou Ethernet.

3. O Plano Lógico da Internet

A Internet funciona graças a regras e sistemas que permitem que milhões de dispositivos comuniquem entre si de forma organizada. Vamos perceber o que acontece quando tentas aceder a um site.

3.1 Endereços IP e o Protocolo TCP/IP

Cada dispositivo ligado à Internet tem um endereço IP, a sua "morada digital". Para garantir que todos comunicam corretamente, existem regras chamadas protocolos. O principal é o TCP/IP, que define como os dados são enviados e recebidos na rede.

3.1.1 Composição de um Endereço IP

O formato mais comum é o IPv4, que consiste em quatro grupos de números separados por pontos:

Exemplo: 192.168.1.1

  • Cada número varia entre 0 e 255.
  • O IPv4 permite aproximadamente 4,3 mil milhões de endereços únicos.

Devido à limitação de endereços disponíveis, surgiu o IPv6, um formato mais recente, com números muito maiores e separados por dois pontos:

Exemplo: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

  • O IPv6 permite um número praticamente infinito de endereços (3,4 x 1038).

3.1.2 Tipos de IP: Público vs Privado

  • IP Público: É o endereço visível na Internet. É atribuído pelo teu fornecedor de Internet (ISP).
  • IP Privado: Usado dentro da tua rede doméstica. Exemplo típico: 192.168.x.x.

3.1.3 IP Fixo vs Dinâmico

  • IP Dinâmico: A maioria das ligações domésticas usa IPs que mudam periodicamente.
  • IP Fixo: Empresas ou serviços que precisam de um endereço constante usam IPs estáticos.

Certos IPs são reservados, como 127.0.0.1, conhecido como "localhost", que aponta sempre para o próprio computador. Podes ver qual o teu IP em What Is My IP Address.

Graças ao sistema de IPs e ao protocolo TCP/IP, todos os dispositivos conseguem comunicar entre si, seja numa rede doméstica ou na vasta rede mundial que é a Internet.

3.2 DNS, O Tradutor da Internet

Como ninguém decora endereços IP, usamos nomes como google.com. O DNS (Domain Name System) traduz esses nomes para o IP correspondente, permitindo que o teu pedido chegue ao servidor certo.

3.3 O que é um URL

O URL (Uniform Resource Locator) indica exatamente o recurso que queres aceder num servidor.

Exemplo:

https://www.exemplo.com/pasta/pagina.html

  • https://, Protocolo de comunicação.
  • www.exemplo.com, Nome do domínio traduzido pelo DNS.
  • /pasta/pagina.html, Localização específica no servidor.

3.4 HTTP e HTTPS, Como Comunicas com o Site

O protocolo HTTP permite a troca de informação entre o teu browser e o servidor. Quando vês HTTPS, significa que essa comunicação está encriptada, protegendo os teus dados.

3.5 Pacotes de Dados, Como a Informação Viaja

A informação na Internet é dividida em pacotes. Cada pacote contém parte da informação, o endereço IP de destino e instruções para ser reconstituído ao chegar. Graças ao protocolo TCP/IP, mesmo que sigam caminhos diferentes, todos os pacotes chegam corretamente ao destino.

4. A Internet Visível e Oculta

4.1 Surface Web

É a parte da Internet que todos usamos diariamente, acessível através de motores de busca. Inclui sites públicos como o Google, YouTube ou Wikipédia.

4.2 Deep Web

A Deep Web é composta por conteúdos que não aparecem nas pesquisas: emails, contas bancárias, bases de dados privadas, plataformas escolares, etc. A Deep Web é essencial e legítima, é onde estão os teus dados protegidos por passwords.

4.3 Dark Web

A Dark Web é uma pequena parte da Deep Web que exige softwares especiais (como o Tor) para aceder. É usada por jornalistas, ativistas e pessoas que precisam de anonimato. No entanto, também é conhecida por alojar mercados ilegais e outras atividades perigosas.

Importante: A Dark Web não deve ser explorada sem conhecimento. Não é um "site secreto", é uma rede paralela que exige responsabilidade.

5. Velocidade da Internet e Ligações

Quando contratamos um serviço de Internet, vemos valores como 100 Mbps ou 1 Gbps. Estes números representam a velocidade máxima teórica de transferência de dados.

5.1 Unidades de Velocidade

  • Kbps (Kilobits por segundo): 1 Kbps = 1.000 bits por segundo. Muito lento, usado em tecnologias antigas.
  • Mbps (Megabits por segundo): 1 Mbps = 1.000 Kbps. Unidade mais comum nas casas.
  • Gbps (Gigabits por segundo): 1 Gbps = 1.000 Mbps. Fibra ótica avançada.
  • Tbps (Terabits por segundo): 1 Tbps = 1.000 Gbps. Usado apenas em infraestruturas globais.

Nota importante: A velocidade da Internet é medida em Megabits, mas os ficheiros que descarregas são medidos em Megabytes. Como 1 byte = 8 bits:

8 Mbps = 1 MB/s

Ou seja, se tens uma ligação de 100 Mbps, a tua velocidade máxima real de download será cerca de 12,5 Megabytes por segundo.

5.2 Fatores que Influenciam a Velocidade

  • Tipo de ligação:
    • Ethernet: Mais rápida e estável.
    • Wi-Fi: Mais prática, mas sujeita a interferências.
  • Distância ao router
  • Número de dispositivos ligados
  • Limitações do teu equipamento

5.3 Download vs Upload

Nem sempre a velocidade de download (receber dados) é igual à de upload (enviar dados). Em muitos contratos, o upload é bastante inferior, o que afeta videoconferências ou envio de ficheiros grandes.

5.4 Como Testar a Tua Velocidade

Podes usar ferramentas como speedtest.net para verificar a tua velocidade real. Testa sempre por cabo para resultados mais fiáveis.

6. Desafio da Sessão

1. Explora o mapa dos cabos submarinos e encontra a ligação de Portugal.

2. Faz um teste de velocidade em speedtest.net.

3. Pesquisa o teu endereço IP e descobre se estás a usar IPv4 ou IPv6.
