pyenv-check-reqs

Jun 2025 • Concluído

Palavras-chave: AUR (Arch User Repository), Python, Pyenv

Resumo: Não estás farto de criar um ambiente Python para um projeto em desenvolvimento, ver um requirements.txt e não ter a certeza de que versões do Python conseguem sequer executar as dependências? Acabou. Apresento o pyenv-check-reqs, um simulador de pipx/pyenv que tenta executar versões do Python da 3.8 até à mais recente e indica as versões que executam o teu projeto.
