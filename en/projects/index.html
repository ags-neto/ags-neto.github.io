pyenv-check-reqs

Jun 2025 • Finished

Keywords: AUR (Arch User Repository), Python, Pyenv

Summary: Aren't you tired of creating a python environment for a developing project, seeing that requirements.txt and not being sure which python versions best even runs its dependencies? No more! Introducing, pyenv-check-reqs, a pipx, pyenv simulator that tries to run python versions from 3.8 to most recent, and outputs you the versions that runs your projects!
